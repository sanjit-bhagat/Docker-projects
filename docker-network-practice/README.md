# Docker Networking - Custom Bridge Network

## Overview

This project demonstrates Docker networking by creating a custom bridge network and connecting multiple Ubuntu containers.

## Features

- Create a custom bridge network
- Connect multiple containers
- Verify communication using IP addresses
- Verify communication using container names
- Inspect Docker network configuration
- Understand Docker DNS

## Commands Used

- docker network create
- docker network ls
- docker network inspect
- docker inspect
- docker exec
- ping

## Outcome

Successfully configured container-to-container communication using a user-defined Docker bridge network and verified connectivity through both IP addresses and Docker's built-in DNS.
