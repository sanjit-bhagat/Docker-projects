# 🐳 Docker Container Management

## Overview

This project is part of my Docker learning journey. It focuses on practicing Docker container management operations using the Ubuntu image.

## Objectives

- Create Docker containers
- Start and stop containers
- Restart containers
- Rename containers
- Inspect container details
- Execute commands inside a running container
- Monitor container resources
- Remove containers

## Docker Concepts Practiced

- Docker Images
- Docker Containers
- Detached Mode
- Container Lifecycle
- Docker CLI

## Commands Practiced

- `docker pull`
- `docker run`
- `docker ps`
- `docker ps -a`
- `docker start`
- `docker stop`
- `docker restart`
- `docker rename`
- `docker inspect`
- `docker stats`
- `docker exec`
- `docker rm`

## Learning Outcome

Through this practice, I gained hands-on experience in managing Docker containers and understanding the complete container lifecycle. This exercise helped me become familiar with essential Docker commands that are commonly used in DevOps environments.
